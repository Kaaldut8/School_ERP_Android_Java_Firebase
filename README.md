# School ERP Android Application

A full-stack Android application developed using Java and Firebase.

## Features
- Firebase Authentication (Admin/Student login)
- Role-based access control
- CRUD operations for student records
- Firestore real-time database integration
- Attendance tracking system

## Tech Stack
- Java
- Firebase Authentication
- Firebase Firestore
- Android Studio
- Git

## Architecture
- Modular activity-based structure
- REST-inspired backend logic
- Real-time cloud database
